# 프로젝트 업무 정리 가이드 (Mandates)

이 파일은 `ai/dashboard` 프로젝트의 업무 보고서 자동화 규칙을 정의합니다. 모든 Gemini 에이전트는 이 규칙을 최우선으로 준수해야 합니다.

## 1. 업무 시간 계산 (8시간 법칙)
- **서브 업무:** 업무 내용에 `[HH:MM]` 형태의 시간 태그가 있는 경우 해당 시간을 사용.
- **메인 업무:** 시간 태그가 없는 업무.
- **계산식:** `메인 업무 시간 = (8시간 + 연장시간) - (서브 업무 시간 총합)`
- **다중 메인 업무:** 한 날짜에 메인 업무가 여러 개라면 남은 시간을 균등 배분.

## 2. 엑셀 파일 생성 조건
- **파일 이름:** `업무보고서_업무.xlsx`
- **시트 이름:** `상세업무보고서`
- **날짜 형식:** `yyyy.mm.dd(요일)` (예: 2026.03.31(화))
- **텍스트 정제:** 엑셀 기록 시 업무 내용에서 `[HH:MM]` 태그를 삭제하고 공백 제거.
- **컬럼 순서:** 날짜, 프로젝트, 서브코드, 업무내용, 담당자, 셀, 소요시간, 연장시간_적용

## 3. 데이터 관리 규칙
- **이어 붙이기(Append):** 새로운 데이터는 기존 파일 하단에 추가.
- **중복 제거:** `[날짜, 프로젝트, 업무내용, 담당자]`가 동일한 행은 최신 데이터로 유지.
- **결측치 처리:** 날짜 셀이 비어있을 경우 상단 행의 날짜를 가져와 채움(ffill).
