# 업무 추진 현황 대시보드 프로젝트 상태

## 1. 프로젝트 목적
- 업무보고서(Excel) 데이터를 기반으로 프로젝트별 업무 타임라인 시각화 및 실시간 통계 차트 대시보드 구현.

## 2. 데이터 매핑 규칙 (1:1 매칭)
업무보고서_업무.xlsx 파일을 기준으로 아래 매핑이 적용되어 있습니다.
- **날짜**: Index 0 (YYYY-MM-DD 표준 변환 완료)
- **업무내용**: Index 4 (가장 강조되는 메인 텍스트)
- **프로젝트**: Index 2 (사이드바 및 그룹 헤더 기준)
- **담당자**: Index 5
- **셀**: Index 6
- **소요시간**: Index 7 (HH:MM 형식을 float로 변환, 차트 데이터 기준)

## 3. 대시보드 구성
- **사이드바**: 프로젝트별 리스트(시각적 하이라이트 제공).
- **메인 영역**: 프로젝트별 그룹화 타임라인 (헤더에 총 합계 표시).
- **통계 패널**: Chart.js 기반의 '투입 비중(Doughnut)'과 '인원별 투입시간(Bar)' 차트.
    - 전체 보기 시: 프로젝트별 집계.
    - 프로젝트 선택 시: 담당자별 집계.

## 4. 중요 설정
- **차트 레이아웃**: 영역 밀림 방지를 위해 고정 높이(250px) 및 `chart-container` 래퍼 적용.
- **필터링**: 기간 필터 및 프로젝트 선택 필터는 동시에 동작하며 실시간으로 반영됨.
